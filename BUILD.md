ISTE 422 Group Project Deployments
## Group Dragon

### Using Make
#### Building a JAR
``` make -f Makefile.proj create-jar ```

#### Running the JAR
``` java -jar dragon.jar ```

### Using Gradle
#### Building a JAR
``` gradle jar ```

#### Running the JAR
``` java -jar build/libs/iste422dragon.jar ```

### Using Bash Script
#### Build, Test, Run, JAR
``` ./build.sh ```

#### Running the JAR
``` java -jar dragon.jar ```
