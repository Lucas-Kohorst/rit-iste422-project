## Group Dragon

### Using Make
#### Build
``` make -f Makefile.proj build ```

#### Test
``` make -f Makefile.proj run-tests ```

#### Run
``` make -f Makefile.proj run ```

#### Clean
``` make -f Makefile.proj clean ```

#### Building a JAR
``` make -f Makefile.proj create-jar ```

#### Running the JAR
``` java -jar dragon.jar ```

### Using Gradle
#### Build
``` gradle build ```

#### Test
``` gradle test ```

#### Run
``` gradle run ```

#### Clean
``` gradle clean ```

#### Building a JAR
``` gradle jar ```

#### Running the JAR
``` java -jar build/libs/iste422dragon.jar ```

### Using Bash Script
#### Build, Test, Run, JAR
``` ./build.sh ```

#### Running the JAR
``` java -jar dragon.jar ```
